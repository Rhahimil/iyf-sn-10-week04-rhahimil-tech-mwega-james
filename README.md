# IYF Season 10 - Week 4: JavaScript Fundamentals

## Project Overview
This repository contains the deliverables for Week 4 of the IYF Weekend Academy. The focus of this week was transitioning from static web design to dynamic programming using JavaScript.

### Deliverables:
1.  **Interactive Calculator:** A functional grid-based calculator capable of handling basic math, decimals, and complex operations.
2.  **Student Grade Tracker:** A data-driven application that manages student records using Arrays of Objects and advanced Array methods.
3.  **Daily Challenges:** Implementation of FizzBuzz, String Reversal, and Palindrome checking.

---

## 1. Interactive Calculator
A semantic HTML/CSS calculator built to practice **Functions** and **Control Flow**.

### Key Features:
- **Math Operations:** Addition, Subtraction, Multiplication, Division, Modulus, and Exponents.
- **State Management:** Handles decimal point logic (preventing multiple decimals in one number).
- **Error Handling:** Includes specific logic to prevent "Division by Zero" errors.
- **Semantic UI:** Built using `<main>`, `<section>`, and `<output>` for accessibility.

---

## 2. Student Grade Tracker
A logic-heavy application built to practice **Lesson 8: Arrays, Objects & Data Manipulation**.

### Technical Implementation:
- **Data Structure:** Uses an `Array` of `Objects` to store student profiles.
- **Array Methods:** - `.map()` to generate names.
  - `.filter()` to identify struggling students (Avg < 70).
  - `.reduce()` to calculate grade point averages (GPA).
  - `.find()` to locate specific student records.
- **Object Iteration:** Uses `Object.entries()` to loop through subject-specific grades and generate report cards.

---

## 3. Daily Challenges (Lesson 7 & 8 Exercises)
Included in the `main.js` file are solutions to the following logic puzzles:
- **FizzBuzz:** Loop-based logic for multiples of 3 and 5.
- **String Reversal:** Practicing string-to-array conversion.
- **Palindrome Checker:** String manipulation and case-insensitive comparison.
